AI Mental Health Companion

Project Overview:
This project is an AI-based Mental Health Companion designed to help students who are going through stressful and anxious times.

This system detects the user’s mood using Natural Language Processing (using TextBlob sentiment analysis) and responds to the user’s condition with empathetic and supportive responses.

Features:
- Mood Detection using NLP
- Crisis Keyword Detection
- Supportive Response Generation
- Chat History Tracking
- Mood Analytics (Bar Chart)
- Mood Trend Analysis (Line Graph)
- Beautiful Streamlit UI
- Cloud Deployment

Technologies Used:
- Python
- Streamlit
- TextBlob (NLP Sentiment Analysis)
- Pandas
- Matplotlib

Safety Features:
- Crisis keyword detection
- Helpline guidance display
- Non-diagnostic supportive responses

Future Scope:
- Integration with Large Language Models
- Therapist appointment booking
- Database storage
- Login authentication
- Advanced emotional analytics
